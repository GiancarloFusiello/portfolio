About Me

For the last seven years, I have worked in the IT industry, from providing support for users to infrastructure in universities to financial trading floors. I have been exposed to a lot of technology during my experience and find myself constantly learning about new technologies, which is one of the aspects I enjoy working in IT.

In early 2014, I decided to re-train myself to become a software engineer - something that I have always wanted to do but didn't have the confidence to take up earlier but now I'm determined.

I started with learning Java - learning the basic principles of programming and Object Oriented Programming. I was particularly fond of creating Apps for Android as the tools are free and Google provide a lot of documentation. Once I become confident with programming, I started to learn frontend web technologies such as HTML5, CSS3 and Javascript. Shortly after, I discovered Python and the Django framework. Since discovering Python, I have to say, I'm a huge fan and find myself researching what else I can use Python for.

More recently, I have completed many courses from Codecademy.com, Treehouse.com, CodingForEntrepreneurs.com and Dataquest.io, as well as many Django tutorials such as Django Polls and DjangoGirls. In these courses I have progressed with learning Python and have also become familiar with Django, jQuery, Bootstrap, Git, SQL, AJAX, REST APIs, libraries such as Requests, NumPy, Pandas, scripting using Bash, Bat and PowerShell and with using Windows CMD, Mac OS X and linux terminals.

My goal is make a career out of something I enjoy so that I never have to work another day again.
